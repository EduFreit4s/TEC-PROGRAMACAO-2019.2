1. Faça um programa em C++ que leia três palavras e mostre a palavra com mais caracteres a palavra com menos caracteres ou se todas
possuem a mesma quantidade de caracteres.

2. Você foi contratado para criar um pequeno sistema de controle de clientes em uma startup a sua tarefa é gerenciar o cadastramento
dos novos clientes. Os clientes são divididos em duas categorias: pessoa física e pessoa juridíca. 

	Todos os clientes possuem os seguintes dados:
		código
		nome
		telefone
		cep

	Os clientes do tipo pessoa física possuem os seguintes dados:
		cpf
		rg
		data de nascimento

	Os clientes do tipo pessoa jurídica possuem os seguintes dados:
		cnpj
		inscrição estadual
		nome fantasia

	Utilizando orientação a objetos e herança, crie um programa onde seja possível cadastrar 3 pessoas físicas, 3 pessoas jurídicas
	e salvar os dados separadamente em pessoa_fisica.txt e pessoa_juridica.txt, os dados no arquivo não devem ser sobrescritos.

	Obs.: Os objetos não devem ter valores vazios depois de feito o cadastramento de um cliente.

3. Faça um programa que relacione uma quantidade indefinida de nomes e sobrenomes. Não podem haver sobrenomes repetidos se o usuário
tentar cadastrar um sobrenome repetido, deve ser dada opção de sobrescrever ou não o valor cadastrado.

	Ao final do cadastramento o programa deve exibir todas relações no formato: sobrenome, nome.
	Ex.: 
		de Tal, Fulano
		Skywalker, Leia
		Vader, Darth
		...

4. Faça um programa que leia uma quantidade indefinida de valores inteiros e mostre ao final:
	
	Os valores ordenados.
	A média dos valores.
	A mediana dos valoes.

	Mediana é o valor que separa a metade maior e a metade menor de uma amostra, uma população ou uma distribuição de probabilidade.
	Em termos mais simples, mediana pode ser o valor do meio de um conjunto de dados. No conjunto de dados {1, 3, 3, 6, 7, 8, 9},
	por exemplo, a mediana é 6. Se houver um número par de observações, não há um único valor do meio. Então, a mediana é definida
	como a média dos dois valores do meio. No conjunto de dados {3, 5, 7, 9}, a mediana é  (5+7)/2 = 6.
